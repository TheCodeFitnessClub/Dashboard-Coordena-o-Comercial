API Closum — Sincronização automática

As leads entram automaticamente no ficheiro assim que são criadas no Closum — sem exportações manuais. Quando um comercial actualiza o estado de sequência no Closum, o tempo de resposta é calculado automaticamente.
Sincronização completa
Reavalia a source de todas as leads a partir do Closum — usa isto quando alteraste sources no Closum (ex.: mudar uma lead para Outbound) e queres garantir que ficam refletidas. É mais lenta (mais chamadas à API). Não afeta os tempos de resposta nem outros dados.

Google Calendar — Integração de Tarefas

Quando ativo, ao preencheres o prazo de uma tarefa abre automaticamente um tab do Google Calendar com o evento pré-preenchido — basta clicar Guardar uma vez.
Podes também usar o botão 📅 em qualquer linha de tarefa para abrir o evento manualmente.

Dados & Backup

💾 Guardar no ficheiro embutis os dados directamente no HTML — se abrir o ficheiro noutra máquina ou browser os dados estão lá.

Auto-guardado no browser acontece automaticamente a cada edição (localStorage).

Exportar XLSX gera um workbook com todas as secções para partilha ou arquivo.

Backup JSON exporta os dados brutos — útil para importar numa versão nova do ficheiro.
